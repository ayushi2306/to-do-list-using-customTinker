# 🎨 Modern To-Do List using CustomTkinter

## 📌 Overview

This project is a modern desktop To-Do List application developed using **CustomTkinter**, an enhanced version of Tkinter that provides modern-looking widgets and customizable themes.

The goal of this project was to explore desktop application development while learning how to build visually appealing user interfaces entirely in Python.

---

## ✨ Features

* Modern graphical interface
* Add tasks
* View tasks
* Delete tasks
* Numbered task list
* Input validation
* Responsive layout
* Dark mode interface

---

## 🛠️ Technologies Used

* Python 3
* CustomTkinter
